# Image Reviewer GUI

Install and run the dev instance:

```
> yarn install && yarn start
```

Build for production:

```
> yarn build:mac | yarn build:windows | yarn build:linux
```

## Usage

- <kbd>→</kbd> next image
- <kbd>←</kbd> prev image
- <kbd>Space</kbd> delete image

## Files

- Electron backend - `public/*`
- React frontend - `src/*`
