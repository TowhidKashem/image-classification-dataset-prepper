# Image Reviewer GUI

Install and run the dev server:

```
> yarn install
> yarn start
```

Build the app as a standalone portable on mac:

```
> yarn build
```

## Usage

- <kbd>→</kbd> next image
- <kbd>←</kbd> prev image
- <kbd>Space</kbd> delete image

## Files

- Electron backend - `public/electron/*`
- React frontend - `src/*`
